# Workflow Diagnosis Card

Always return this shape.

## Diagnosis

- code: one value from `reference/diagnosis-taxonomy.md`
- confidence: `high`, `medium`, or `low`
- current state: one sentence describing what Chromatic is doing on this run

## Key evidence

List the strongest facts that support the diagnosis.

## Why this happens

Explain the git, branch, or baseline rule in plain language.

## Next step

Give one exact next artifact or one exact change to make.

## After the fix

Say what should change on the next run.
