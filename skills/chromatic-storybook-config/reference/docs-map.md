# Documentation Topic Map

Installed skills should explain the issue directly and only point a human toward doc topics when useful.

| Diagnosis code | Documentation topic |
| --- | --- |
| `SB_CFG_STORIES_GLOB_MISMATCH` | Story discovery and story file patterns |
| `SB_CFG_FRAMEWORK_OR_BUILDER_MISMATCH` | Framework and builder configuration |
| `SB_CFG_ADDON_CONFLICT` | Addon compatibility and registration |
| `SB_CFG_PREVIEW_RUNTIME_ERROR` | Preview decorators, globals, and shared providers |
| `SB_CFG_STATIC_ASSET_PATH` | Static assets and public directory handling |
| `SB_CFG_PACKAGE_VERSION_DRIFT` | Version alignment across Storybook packages |
| `SB_CFG_LOCAL_BUILD_BROKEN` | Local Storybook build troubleshooting |
