# Documentation Topic Map

Installed skills should explain the issue directly and only point a human toward doc topics when useful.

| Diagnosis code | Documentation topic |
| --- | --- |
| `WF_INVALID_GIT_HISTORY` | Git history requirements and CI checkout depth |
| `WF_PR_EVENT_MISMATCH` | GitHub Actions pull request configuration |
| `WF_MISSING_COMMIT_DETAILS` | CI branch and SHA forwarding |
| `WF_REBASE_BASELINE_CONFUSION` | Rebasing and baselines |
| `WF_NO_ANCESTOR_BUILD` | Branching and baseline behavior |
| `WF_REPLACEMENT_BUILD` | Replacement builds and historic commits |
| `WF_MERGE_QUEUE_BRANCH_REMAP` | Merge queue behavior |
| `WF_MERGE_BASE_NOT_FOUND` | Merge-base calculation and patch builds |
| `WF_PATCH_BUILD_REQUIRED` | Patch builds |
