# Documentation Topic Map

Installed skills should explain the recommendation directly and only point a human toward doc topics when useful.

| Scenario | Documentation topic |
| --- | --- |
| One shared Storybook | Monorepo setup |
| Multiple independent Storybooks | Monorepo setup with multiple projects |
| Nx build orchestration | Build command and output directory setup |
| Turborepo repo-root execution | Working directory and Storybook base directory |
| Shared root lockfile churn | TurboSnap for monorepos |
| `externals` recommendation | TurboSnap externals guidance |
| `untraced` recommendation | TurboSnap untraced guidance and tradeoffs |
